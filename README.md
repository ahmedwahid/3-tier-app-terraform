# 3-tier-app-terraform
This is a project to deploy 3 tier application through Terraform on AWS

## Resources to be deployed:
- Custom VPC
- 2 Subnets (Public)
- 1 Subnet (Private)
- 2 EC2 Instances
- Security Groups
- Elastic IP
- NAT Gateway
- Internet Gateway
- Route Table
- Application Load Balancer
- Apache Webserver
- MySQL DB
